<br> #### Shaper Tutorial 1. Instal the Shaper plugin for Fusion 360. Here is the <a href="https://apps.autodesk.com/FUSION/en/Detail/Index?id=3662665235866169729&os=Win64&appLang=en">link </a>. Once installed, run the installer to acutaly install the plugin. Follow the downloaders instructions. Windows defender will warn you not to run the program. ![Windoswarn](./windowswarn.png) If you click more info, you can still run it. ![windowsrun](./windowsrun.png) 2. Desgin Design a 2d object in fusion. Then extrude the object a small amount. Then click utlities and you will see the fusion plugin on the left side of the ribbon. If you dont see it try restarting fusion. If you do see it follow the plugin's instructions and export the file as an svg to a usb useing the plugin. ![fusion tutorial](./fusiondemo.gif) 3. Set up The shaper uses tape to locate it self and cut. Seting up an area with tape is nessiary to useing the machine. The tape does not stick to obs, but if you are doing a small print you can get away with no tape over where you are cuting. The machine uses two cameras. One over the spindle and one looking infornt of the machine. It uses the one looking over the machine to postion it self. Make sure to measure the depth of the material you are cutting. In the lab there is a sacrifce board right now. If there is no sacrfice board it is probaly a really good idea to put one down so you dont destroy the surface you are cutting on. Make sure to bolt down the material you are cutting. I have only cut wood so far. ![setup](./workspace.jpg) 3. Scan Plug the Shaper in. Before cutting you must make a new scan of work area. It scans using the camera in front of the machine. Sometimes you might need to scan a little bit off the tabel in order to see the wood while cutting. ![scan](./scan.gif) 4. Device Plug in the usb to the usb port. Then click import. Find your file then place it using the intuitave placing system. There are scalling options too. Place the design on the object on the surface you are cutting on. Then click cut. Now, rember the depth of the material enter that into the depth setings in the top left corner of the cutting page. ![cutmenu](./cutmenu.gif) Then turn on the spindle. Then click the green button. Then starting moving the machine along the guided route. Tips and Tricks Alwats point thte front facing camera twards the tape that is how the machine orients itself.
